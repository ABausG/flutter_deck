## Installation 💻

**❗ In order to start using flutter_deck you must have the [Flutter SDK][flutter_install_link] installed on your machine.**

Add `flutter_deck` to your `pubspec.yaml`:

```yaml
dependencies:
  flutter_deck:
```

Install it:

```sh
flutter packages get
```
